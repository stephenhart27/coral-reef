# Coral Reef Explorer

An interactive 3D coral reef experience built with **Three.js**, designed to run directly in the browser.

Explore different coral species by selecting them in the scene or via the UI, view contextual information, and smoothly orbit around points of interest.

Live demo is hosted via **GitHub Pages**.

---

## Features

- Interactive **Three.js** scene
- Click / tap corals to view detailed information
- UI buttons for quick coral selection
- Smooth animated camera focus using `OrbitControls`
- Unified interaction model (scene + UI)
- Mobile & touch friendly

---

## Controls

### Desktop
- **Mouse drag** — rotate view
- **Scroll wheel** — zoom
- **Click coral** — focus and view info

### Mobile
- **One finger drag** — rotate view
- **Pinch** — zoom
- **Tap coral or UI button** — focus and view info

---

## Tech Stack

- [Three.js](https://threejs.org/)
- GLTFLoader
- OrbitControls
- EffectComposer (post-processing)
- Vanilla HTML / CSS / JavaScript
- Models sourced from SketchFab, including specimens from the Smithsonian Institiution
